Algorithm Implemented


Initialization (init function):

  • Initializes the grid for the game with dimensions COLUMNS and ROWS.
  • Sets the initial position, length, and direction of the snake.
  • Initializes the game window with OpenGL and GLUT settings.

Main Function (main):

  • Initializes GLUT and creates a window for the game.
  • Sets up display, reshape, timer, and keyboard callback functions.
  • Enters the GLUT main loop, where the program continues to run until the user closes the window.

Display Function (display_callback):

  • Clears the color buffer.
  • Draws the grid using the drawGrid function.
  • Draws the snake using the drawSnake function.
  • Draws the food using the drawfood function.
  • Swaps the display buffers.

Reshape Function (reshape_callback):

  • Sets up the viewport and projection for the game window.

Timer Function (timer_callback):

  • Calls the display_callback function at regular intervals to update the game screen.
  • Uses GLUT timer functions for the specified frames per second (FPS).

Keyboard Callback Function (keyboard_callback):

  • Handles keyboard input to change the direction of the snake.

Grid Drawing Functions (drawGrid and unit):

  • Draws the game grid, distinguishing the border with a thicker line.
  • The unit function is a helper function for drawing individual units of the grid.

Snake Drawing Function (drawSnake):

  • Updates the position of the snake based on its direction.
  • Handles collisions with the game borders and the snake itself.
  • Checks for collisions with food, increments the snake length, and generates new food.

Food Drawing Function (drawfood):

  • Randomly positions food on the grid.
  • Draws the food on the screen.

Random Function (random):

  • Generates random positions for the food within the game grid.

Command to Run the code:

1.) g++ new_st.cpp gam.cpp -o snake_game -lglut -lGL -lGLU
2.) ./snake_game
